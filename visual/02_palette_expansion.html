Key Concepts — Palette Expansion
What is an Expansion?
When greedy coloring reaches vertex v and all colors 1…k are already used by neighbors of v, we must introduce color k+1. That is one palette expansion.
p(G) — the Key Invariant
p(G) is the minimum number of expansions over all possible vertex orderings. It is a graph invariant — it does not depend on which ordering achieves the minimum.
The Main Theorem
χ(G) = 1 + p(G) for every graph G. The "+1" accounts for the initial color; each expansion introduces exactly one new color that is strictly necessary.
Why C₅ needs 3 colors
C₅ is an odd cycle. Any 2-coloring must alternate, but after going around 5 vertices the last is adjacent to both ends — forcing a third color. So p(C₅) = 2 always.
