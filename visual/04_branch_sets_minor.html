Definition 3.4 — Graph Minor

Branch Sets and the Kk Minor

How contracting connected subsets reveals a complete graph hidden inside G

Definition 3.4  A graph G contains Kk as a minor if there exist k pairwise disjoint, non-empty, connected subsets B1, B2, …, Bk ⊆ V(G) such that for every pair i ≠ j there is at least one edge between Bi and Bj. Contracting each Bi to a single vertex gives Kk.

Hadwiger's Conjecture: if the chromatic number χ(G) = k, then G contains Kk as a minor.
Equivalently: palette-expansion number p(G) = k − 1  ⟹  G has Kk minor. The branch sets Bi are constructed explicitly in Construction 8.1 (see File 5).
