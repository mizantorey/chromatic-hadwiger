Step 5 — The Discovery: χ(G) = 1 + p(G)
Every time greedy coloring forces a new color, that is a palette expansion. Count them all up. The chromatic number equals ONE plus the number of unavoidable expansions.
χ(G) = 1 + p(G)
The chromatic number equals one (first color used) plus the number of palette expansions
